# MicroDocs Core library
Core library used across other MicroDocs Typescript projects

## Development
### Setup
```
npm install
npm run link
npm run watch
```
### Test
```
npm run test
```
### Build
```
npm run prepublish
```

### Publish
```
npm publish dist --registry https://nexus-mxtf.maxxton.com/repository/npm-registry/
```
